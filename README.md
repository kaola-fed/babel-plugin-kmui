# babel-plugin-kmui

[![build status][build-status-image]][build-status-url]
[![coverage][coverage-image]][coverage-url]

import KMUI component on-demand

## Usage

```js
// .babelrc
{
  "plugin": ["kmui"]
}
```

## Options

### libraryName

Type: `String`

Default: `kmui`
