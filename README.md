# babel-plugin-kmui

[![build status][build-status-image]][build-status-url]

import KMUI component on-demand

## Usage

```js
// .babelrc
{
  "plugin": ["kmui"]
}
```

## Options

### libraryName

Type: `String`
Default: `kmui`
